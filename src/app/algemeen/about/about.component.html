Over de BierWijnApp:
De BierWijnApp is een leerproject dat is opgezet door drie Kadaster Java-trainees in 2018.
Doel van dit leerproject was om binnen het traineetraject dat liep van februari t/m decemeber 2018 extra ervaring
op te doen in verschillende onderdelen van het developersvak. Een overzicht van wat er gebruikt is aan talen/technieken/libraries om deze applicatie op te zetten:

  • Java
  • PostrgresSQL
  • Angular
  • Spring Boot
  • Bootstrap
